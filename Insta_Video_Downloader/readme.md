### Whsiper model sizes
depends on model:  
- tiny → ~75 MB
- base → ~150 MB
- small → ~500 MB
- medium → ~1.5 GB
- large → ~3 GB+

### Command
```
whisper v1.mp4 --model small --language en
```

### Installation of Whisper
```
pip install openai-whisper
```

Whisper requires dependency of ffmpegusing (using admin priv)
```
choco install ffmpeg
```
